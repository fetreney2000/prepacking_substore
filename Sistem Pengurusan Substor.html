Bantuan

Pengenalan

Sistem ini mengurus inventori prabungkus untuk Substor Hospital Keningau. Ia membantu mengira tahap stok, menjejaki pesanan, dan menghasilkan laporan.

Pengiraan

  • AWU (Penggunaan Mingguan Purata) = Purata penggunaan 3 bulan / 4.33
  • Min = AWU x Minggu Minimum
  • Penimbal = AWU x Minggu Penimbal
  • Maks = AWU x Minggu Maksimum

Status Stok

  • OK Stok melebihi tahap penimbal
  • Rendah Stok antara min dan penimbal
  • Kritikal Stok di bawah min
  • Kehabisan Stok sifar

Pintasan Papan Kekunci

  • Arrow keys dalam jadual: tatal
  • Ctrl+F: Fokus medan carian
